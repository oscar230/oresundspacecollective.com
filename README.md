# Øresund Space Collective
This is the website of [Øresund Space Collective](oresundspacecollective.com), based on a copy of `oresundspacecollective.com` from [archive.org](archive.org).

## Develop
### Old static site
```bash
npx http-server src/
```
### New Jekyll site
```bash
cd osc
bundle exec jekyll serve
```
